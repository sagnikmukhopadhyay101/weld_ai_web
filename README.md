Weld AI Web is an AI-powered web application that detects welding defects from images using computer vision.  
The system allows human feedback so the model can improve over time.

## Features
- Upload welding images
- Detect defects using a trained AI model
- Shows defect status clearly
- Human feedback option for wrong predictions
- Designed for real-world industrial use

## Tech Stack
- Python
- Streamlit
- YOLO (Ultralytics)
- OpenCV
- PyTorch
